# EngeekerOJ

## 书写规范

- **一级标题（`#`）**：用于写**算法名称**。
- **二级标题（`##`）**：用于写**问题描述**、**算法步骤** 和 **代码模板** 等内容。

### 示例格式：

````markdown
# 数位和

## 问题描述

给定一个正整数 $x$，求出它各个数位上的数字之和。

## 算法步骤

1. 初始化一个变量 `sum = 0`；
2. 当 $x > 0$ 时：
   - 将 $x$ 的个位数加到 `sum` 上；
   - 用整除 `x = x // 10` 去掉个位；
3. 输出 `sum`。

## 代码模板

```python
x = int(input())
total = 0
while x > 0:
    total += x % 10
    x //= 10
print(total)
```
````
